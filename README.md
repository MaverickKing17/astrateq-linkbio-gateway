## 🔗 Technical Setup & Environment

This is a modern **full-stack React application** running on **Vite** and styled with **Tailwind CSS**.

### Scripts Reference

*   `npm run dev` — Boots the local Vite server.
*   `npm run build` — Compiles state assets for production distribution.
*   `npm run lint` — Validates React 19 type-safety across the environment.
